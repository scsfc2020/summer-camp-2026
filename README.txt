【檔案清單】
📄 index.html              ← 網站主檔（打開就能看到網站）
📁 images/                 ← 所有圖片
   ├─ logo.png             ← 您的原始 LOGO
   ├─ logo-white.png       ← 白色 LOGO（深色背景用）
   ├─ logo-black.png       ← 黑色 LOGO（淺色背景用）
   ├─ favicon.png          ← 瀏覽器分頁圖示
   ├─ hero.jpg             ← 🔄 主視覺（佔位圖，可替換）
   ├─ training-1.jpg       ← 🔄 訓練 1（佔位圖，可替換）
   ├─ training-2.jpg       ← 🔄 訓練 2（佔位圖，可替換）
   ├─ training-3.jpg       ← 🔄 訓練 3（佔位圖,可替換）
   ├─ banqiao.jpg          ← 🔄 板橋場地（佔位圖，可替換）
   ├─ tianmu.jpg           ← 🔄 天母場地（佔位圖，可替換）
   ├─ beida.jpg            ← 🔄 北大場地（佔位圖，可替換）
   ├─ coach.jpg            ← 🔄 教練指導（佔位圖，可替換）
   └─ game.jpg             ← 🔄 比賽場景（佔位圖，可替換）

【快速開始】
1. 下載整個資料夾，保持資料夾結構完整
2. 雙擊 index.html 用瀏覽器打開
3. 就能看到完整網站！

【發布網站前需要替換的內容】
在 index.html 中搜尋並替換：
  YOUR_GOOGLE_FORM_URL     → 主報名表單連結
  YOUR_KOREA_SURVEY_URL    → 韓國意願調查連結

（LINE ID 已經是您的 @scsfc2020，不需要改）

【替換照片方法（超簡單）】
您要替換哪張照片，只要：
1. 準備新照片，命名成原本的檔名（如 hero.jpg）
2. 放進 images/ 資料夾（覆蓋原檔案）
3. 重新整理網站 → 新照片就顯示了！

【照片建議尺寸】
- hero.jpg       1920 × 1080（主視覺，要最大張）
- banqiao.jpg    800 × 600（場地）
- tianmu.jpg     800 × 600（場地）
- beida.jpg      800 × 600（場地）
- training-*.jpg 1200 × 800（訓練場景）
- coach.jpg      1200 × 800（教練）
- game.jpg       1200 × 800（比賽）

【照片壓縮工具（免費）】
- tinypng.com（拖進去自動壓縮）
- squoosh.app（Google 提供）

【部署上線方式】

方法 1：GitHub Pages（推薦、免費）
1. 註冊 GitHub 帳號
2. 建立 repo，上傳整個資料夾
3. Settings → Pages → 啟用
4. 取得網址：yourname.github.io/scsfc-camp2026

方法 2：網域 + 虛擬主機（專業）
1. 買網域（約年費 $500-1,500）
2. 租虛擬主機（約月費 $300-600）
3. FTP 上傳整個資料夾

方法 3：請工程師協助部署
